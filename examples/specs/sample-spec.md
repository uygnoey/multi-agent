# 기획서: 팀 태스크 보드 (TaskBoard)

작은 팀이 할 일을 등록하고 진행 상태를 관리하는 웹 서비스.

## 목표
- 사용자가 로그인하여 자신의 태스크를 관리한다.
- 태스크를 칸반 형태(To Do / Doing / Done)로 본다.

## 핵심 기능
- 사용자 인증 (이메일/비밀번호 회원가입·로그인, 세션 발급)
- 태스크 CRUD (제목·설명·상태·마감일 생성/조회/수정/삭제)
- 칸반 보드 뷰 (상태별 컬럼, 드래그로 상태 변경)
- 태스크 검색·필터 (상태·마감일·키워드)

## 비기능 요구
- API 응답 200ms 이내(로컬 기준), 입력 검증·기본 에러 처리.
- 비밀번호 해시 저장, 인증 토큰은 만료 처리.

## 기본 스택 (아키텍트 재정의 가능)
- Backend: FastAPI (Python)
- Frontend: React + Vite (TypeScript)
- DB: SQLite (운영 시 Postgres 이행 고려)
